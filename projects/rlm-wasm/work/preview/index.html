03-l1-dsl - Level 1 DSL Overview 17.93s

clips/03-l1-dsl.mp4 + audio/l1-overview.wav

R L M Level one uses built in D S L commands for simple text operations. The language model selects operations like find, regex, or count. No code generation, just instant pattern matching and line counting.

Visualization Demo - L1 DSL Web UI 47.15s

vhs/l2-visualization-narrated.mp4

Q1: The R L M Visualizer lets you run queries on sample data. Here we load a Count ERROR lines example with one hundred log entries.
Q2: Clicking Run R L M Query starts the iterative process. The Output tab shows live progress as the language model works through the problem.
Q3: Each iteration executes D S L commands. First count lines, then regex to find errors. The language model refines its approach based on intermediate results.
Q4: Final answer: twenty error lines. Three iterations completed in seven point one seconds. No code generation required, just D S L pattern matching.

04-l2-wasm - Level 2 WASM Overview 19.40s

clips/04-l2-wasm.mp4 + audio/l2-overview.wav

R L M Level two uses web assembly for complex processing. A coder language model writes Rust code that compiles to web assembly and runs in a secure sandbox. Map reduce handles aggregation and statistics on large datasets.

L2 WASM Web UI Demo - Unique IPs ~60s

Setup: Level two web assembly handles complex analysis. Here we select Count unique I P addresses, which uses map reduce with a unique combiner on two hundred log lines.
Run: Clicking Run R L M Query starts iteration one. The language model analyzes the query and decides to use rust wasm map reduce with a unique combiner.

Result: Eight unique I P addresses found. Two iterations in twenty six seconds. The timeline shows step one used web assembly map reduce, step two returned the final answer.

WASM Limitations ~12s

clips/08-limitations.mp4 + audio/limitations.wav

WASM has trade-offs. No unicode support, only ASCII. Memory is limited to 64 megabytes. And only a subset of Rust compiles to WASM. These limits enable safe sandboxed execution.
