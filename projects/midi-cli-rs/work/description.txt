A command-line tool for generating procedural music, built with Rust and FluidSynth.

MIDI CLI generates background music from six mood presets: jazz, upbeat, ambient, calm, suspense, and eerie. Specify duration, tempo, key, and intensity. The tool outputs MIDI files or renders audio directly using a SoundFont.

The web interface connects to a local REST server, letting you create and preview presets without touching the command line. The melody editor goes further‚Äîcompose note by note, shift octaves, change durations, and pick from dozens of instruments.

This is personal software via vibe coding‚Äîbuilt for AI agents that need background tracks, but useful for anyone who wants quick procedural music.
